# Online Swatting Awareness Website

A professional, responsive website dedicated to raising awareness about online swatting, supporting victims and survivors, and providing educational resources about this serious cybercrime.

## 📋 Features

- **Professional Design** - Modern, clean interface with gradient headers
- **Fully Responsive** - Works on desktop, tablet, and mobile devices
- **6 Main Sections:**
  - Home - Hero section with call-to-action
  - Online Swatters - Definition and information about perpetrators
  - Victims & Survivors - Support resources and safety tips
  - Crime Impact - Effects on public safety and communities
  - Related Laws - Federal and state legislation
  - Help & Resources - Hotlines, legal assistance, and support services

## 🚀 Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. Or deploy to Netlify, Vercel, or GitHub Pages

## 📁 Files

- `index.html` - Main website structure
- `styles.css` - Responsive styling
- `script.js` - Interactive functionality
